

#### Python 的模拟命令行程序

---

##### 安装运行
安装 poetry 
```shell script
pip install poetry
```
安装需要的 依赖
```shell script
poetry install
```
运行项目 
```
run.bat
```

---

##### Play
运行正常后, 进入一个模拟的命令行

运行 help 命令 或者两下 tab 可以查看 能够支持的命令

功能非常有限, 主要是为了实现游戏类似效果
